# GitHub Pages Setup (Current Active Deployment)

## ✅ Already Configured!

GitHub Pages is **already set up and working**. No additional configuration needed!

## How It Works

1. **PR Preview**: Open a PR → Auto-deploys to `https://prabdev01.github.io/2048/pr-NUMBER/`
2. **Production**: Merge to master → Auto-deploys to `https://prabdev01.github.io/2048/`
3. **Cleanup**: Close PR → Preview auto-deleted

## Enable GitHub Pages (One-Time Setup)

1. Go to **Settings** → **Pages**
2. Source: **Deploy from a branch**
3. Branch: **gh-pages** / **(root)**
4. Click **Save**

That's it! ✅
